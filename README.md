# Metrics Service — сервис сбора метрик и алертинга

## Описание

Система сбора метрик состоит из агента и сервера. Агент собирает метрики с хоста и отправляет их на сервер с настраиваемой периодичностью. Сервер хранит метрики и предоставляет HTTP-ручки для получения данных по одной или всем метрикам.

---

## Функциональность

### Агент

- Сбор системных метрик (CPU, память, диск, сеть и др.)
- Настраиваемая частота сбора метрик
- Настраиваемая частота отправки на сервер
- Конфигурация через env или флаги командной строки

### Сервер

- Прием и хранение метрик
- HTTP-ручки для получения метрик
  
---

## Архитектура и ключевые решения

- **Агент-серверная архитектура** — агент собирает метрики, сервер хранит и отдает
- **Гибкая настройка** — частота сбора и отправки задается через env или флаги
- **REST API** — HTTP-ручки для доступа к метрикам
- **Поддержка различных типов метрик**

---

## Результат

- Автоматический сбор системных метрик без участия пользователя
- Гибкая конфигурация под разные сценарии использования
- Простой HTTP-доступ к метрикам для интеграции с системами мониторинга
- Возможность использования как основы для алертинга
